# Todyapp - Full-Stack Swift To-Do Application 🚀

Hello! This is **Todyapp**, a complete task management system I built using **Swift** for both Backend and Frontend. 

**Design inspiration:** https://www.figma.com/design/UJUFP9MIpkmXmRVXV9ixlB/Todyapp---Todo-list-mobile-app--Community-?node-id=582-5288&t=QJwbU66X3F1yhRxD-0

### 🛠 Technologies Used

**Backend (Vapor Framework):**
- Written in **Swift 6**.
- **PostgreSQL** database with Fluent ORM.
- **JWT (JSON Web Token)** for secure user authentication.
- **Bcrypt** for password hashing.
- **Eager Loading** for relational data management.

**Mobile (SwiftUI):**
- **MVVM** architecture.
- **async/await** for modern networking.
- **KeychainAccess** for secure token storage.
- **NavigationStack** with a centralized **Route** system.
- **Custom UI Components** and Splash Screen.

### 🌟 Key Features
- **Secure Auth:** Users can register and login with encrypted passwords.
- **CRUD Operations:** Create, Read, Update, and Delete tasks directly linked to your account.
- **User Sync:** Every task is assigned to a specific user via database relationships.
- **Modern UX:** Smooth transitions, adaptive UI, and onboarding screens.
